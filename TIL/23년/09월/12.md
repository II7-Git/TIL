# 23.09.12 TIL

## 취업준비

오늘은 저번에 서류 합격한 회사의 온라인 코딩 테스트를 보았었다. 어떤 식으로 문제가 나올지 잘 몰라서 일단 준비해놓은 것을 믿자 하고
전체적인 배웠던 것들을 다시 한 번 복습하면서 시간을 보냈다. 자세한 시험 내용을 적을 수는 없지만 간단한 피드백을 적으면서 스스로 방향성을 잡으려고 한다.

### 아쉬운 점

1. 공부했던 분야가 아닌 부분들에서 나온 문제에 대해서 대책이 없었다는 점이었다. 앞으로는 게임 개발에 관련해서 좀 더 넓은 분야에서 공부를 계속 해야겠다고 생각이 들었다.
2. C++의 좀 더 다양한 예를 공부해야겠다. 기본적인 사용은 꾸준히 하고 있는 언어기에 익숙하고 자신있다 생각했는데 자주 사용하지 않는 좀 더 C++의 깊숙한 질문이 나오니 스스로 알 것 같은데 당황하게 됐던 것 같다. 그래서 C++에 대해서 좀 더 다양한 공부를 해야겠다.
3. CS를 좀 더 탄탄하게 공부해야겠다.

### 좋았던 점

1. 코딩 문제 자체에 대해서는 좀 더 자신감이 생겼다.
2. 회사가 어떠한 분야와 유형의 문제를 내는지 실전으로 경험할 수 있었다.
3. 그래서 공부할 방향을 다시 한번 설정할 수 있었다.

### 마무리

오늘은 시험 준비를 하느라 시간을 전부 소비했던 하루지만 그래도 나름대로 얻어가는게 많았던 시험이었다는 생각이 들었다. 결과가 어떻게 됐든 이번 경험을 토대로 좀 더 나은 개발자가 될 수 있으면 좋겠다.
